
  # Avant-Garde Portfolio Website

  This is a code bundle for Avant-Garde Portfolio Website. The original project is available at https://www.figma.com/design/yTkHvcx1kdmKK6pyag61pY/Avant-Garde-Portfolio-Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

## Deploying to GitHub Pages

- Production build outputs to `docs/` (see `vite.config.ts`).
- CI deploys via `.github/workflows/deploy-pages.yml`.

Enable it once:

- GitHub repo → **Settings** → **Pages**
- **Build and deployment** → Source: **GitHub Actions**
